## Executive Summary

This release introduces a complete RevealJS presentation system with three-mode theme support (light/dark/vibrant), component-based slide decks with Three.js particle sphere illustrations, and comprehensive markdown slide rendering with GFM table support.

## Key Features

### 1. RevealJS Presentation System

A complete slides system adapted from hypernova-site for Dark Matter's three-mode theme architecture.

**Layouts:**
- **BoilerPlateHTML.astro** — Minimal HTML boilerplate without header/footer for immersive presentations
- **OneSlideDeck.astro** — RevealJS 4.5 wrapper with full theme integration
- **MarkdownSlideDeck.astro** — Markdown-to-slides converter with remark-gfm for GFM tables

**Components:**
- **SlidesControlButtons.astro** — Control bar with three-mode toggle (sun/moon/star icons)
- **TitleSlidePreviewCard.astro** — Preview cards for slides index page

**Pages:**
- `/slides/` — Auto-discovers presentations from all sources
- `/slides/[...slug]` — Dynamic routing with static prerendering

## Technical Details

### Data Layer

**componentDecks.ts:**
- Registry for component-based presentations
- Metadata: title, description, component name, author, date, tags
- Cover image and share image support for social sharing

**markdownDecks.ts:**
- Auto-loader for markdown presentations from content collection
- Stores raw markdown content as strings for prerender compatibility
- Supports frontmatter metadata extraction

### Content Collection

**slides.config.ts:**
- Passthrough schema allowing flexible frontmatter
- Supports: title, description, author, date, tags, coverImage, shareImage

### Static Prerendering

All slide routes use `export const prerender = true` for:
- SSR compatibility with Vercel adapter
- Static generation at build time
- Optimal performance

### Keyboard Navigation

- Arrow keys: Navigate slides
- F: Fullscreen mode
- S: Speaker notes
- ESC: Exit fullscreen
- B: Black screen pause
